개인정보 처리방침

시행일: 2026-07-04

Midnight Tracer(이하 "서비스")는 회원가입 없는 익명 커뮤니티입니다. 본 방침은 서비스가 어떤 정보를 어떻게 처리하는지 설명합니다.

1. 수집하는 정보

• 기기 식별자(device_id): 익명 작성·삭제·차단을 위해 자동 생성된 무작위 문자열
• 작성 콘텐츠: 게시글, 댓글, 사진·영상(선택), 닉네임
• 위치 정보: 거리 계산, 위치 기반 탐지 도구(심령 레이더 등) 실행에 현재 위치를 사용합니다. 지도 탐색에는 사용되지 않습니다. 위치 기반 계산은 기기 내부에서 처리됩니다. 심령 스팟 글은 위치를 비공개로 서술하는 방식으로 작성되며, 좌표는 서버에 저장되지 않습니다
• 마이크(음성): 일부 탐지 도구는 소리 분석을 위해 마이크를 사용하며, 해당 분석 처리는 기기 내부에서만 이뤄지고 외부로 전송되지 않습니다. 다만 수동 화면 녹화 기능을 사용하면 마이크로 녹음된 음성이 녹화된 영상 파일에 포함되며, 사용자가 해당 영상을 업로드·공유하도록 선택하면 그 음성도 함께 전송될 수 있습니다. 고스트박스·심령 보드의 음성 인식 기능은 기기 OS(Apple·Google)의 음성인식 서비스를 이용하며, 기기·언어 설정에 따라 음성 데이터가 해당 OS 제공업체 서버로 전송되어 처리될 수 있습니다
• 서비스 이용 기록: 익명 통계 목적 (예: 카테고리별 조회수)

서비스는 이름·이메일·전화번호 등 직접 연락 가능한 식별 정보를 수집하지 않습니다. IP 주소는 앱 서버 레벨에서 직접 저장하지 않지만, Supabase·Cloudflare 등 인프라 제공업체는 보안·장애 대응 목적의 접속 기록을 자체 정책에 따라 처리할 수 있습니다.

2. 사용 목적

• 서비스 제공(글/댓글 작성·표시·삭제)
• 어뷰즈 방어(rate limit, 신고 처리, 자동 숨김)
• 익명 통계(인기 카테고리, 활성 사용자 수)

수집한 정보는 위 목적 외의 용도로 사용되지 않습니다.

3. 보존 기간

• device_id: 사용자가 앱 데이터를 삭제할 때까지. 다만 이용 제한(밴) 조치가 내려진 기기 식별자는 재가입을 통한 제한 우회를 방지하기 위해 앱 데이터 삭제와 무관하게 별도 보관되며, 조치 해제 시 삭제됩니다
• 게시글/댓글: 작성자가 삭제하면 즉시 비공개 처리됩니다. 부정 이용 방지·법적 분쟁 대응 등을 위해 서버에 일정 기간 보관된 뒤 삭제될 수 있습니다

4. 파기 절차 및 방법

서비스는 개인정보의 보유 기간이 경과하거나 처리 목적이 달성된 경우 지체 없이 해당 정보를 파기합니다.
• 전자적 파일 형태로 저장된 정보는 복구·재생이 불가능한 기술적 방법으로 삭제합니다
• 이용 제한(밴) 우회 방지 목적으로 별도 보관하는 기기 식별자는 조치가 해제되는 즉시 파기합니다
• 사용자가 앱 내에서 게시글/댓글/사진·영상을 삭제하면 즉시 비공개 처리 후, 부정 이용 방지·법적 분쟁 대응에 필요한 기간이 지나면 서버에서도 파기합니다

5. 자동 수집 장치의 설치·운영 및 거부

서비스는 앱을 최초 실행할 때 익명 작성·삭제·차단을 위해 무작위 기기 식별자(device_id)를 자동으로 생성해 저장합니다. 또한 자동 가입(봇) 방지를 위해 Cloudflare Turnstile이 기기·브라우저 환경 정보를 판별에 사용합니다.

서비스는 웹 브라우저 쿠키를 사용하지 않습니다. 위 자동 수집 장치의 설치·운영을 거부하려면 앱 데이터를 삭제하거나 앱을 삭제할 수 있으며, 이 경우 서비스의 전부 또는 일부를 이용할 수 없습니다.

6. 처리위탁 및 국외 이전

서비스는 아래 국외 인프라 제공업체에 개인정보 처리 업무를 위탁하거나, 서비스 제공 과정에서 국외 이전이 발생할 수 있습니다. 이전은 사용자가 커뮤니티·오류 리포팅·자동가입 방지 기능을 사용할 때마다 수시로, HTTPS 등 보안 통신으로 이뤄집니다.

• Supabase, Inc. (미국, https://supabase.com/privacy): 데이터베이스·인증·익명 계정 관리. device_id, 게시글·댓글·닉네임, 신고·차단·관리용 메타데이터가 이전될 수 있습니다. 보유기간: 본 방침 §3(보존 기간)·§4(파기 절차)에서 정한 기간까지이며, 위탁 업무 종료 또는 이용자의 삭제 요청 시 지체 없이 파기합니다(수탁사 백업 등으로 인한 삭제 지연 가능)
• Cloudflare, Inc. (미국 등, https://www.cloudflare.com/privacypolicy/): R2 이미지·영상 저장, Turnstile 자동가입 방지, 보안·전송 인프라. 업로드 미디어, 미디어 키, 기기·브라우저 환경 신호, 접속 로그가 이전될 수 있습니다. 보유기간: 콘텐츠 삭제 또는 계정 제한 해제 시점까지이며, 이후 지체 없이 파기합니다(수탁사 백업 등으로 인한 삭제 지연 가능)
• Functional Software, Inc. d/b/a Sentry (미국, https://sentry.io/privacy/): 오류·비정상 종료 진단. 기기·OS 정보, 오류 로그, 앱 상태 정보가 이전될 수 있습니다. 이름·이메일·전화번호 등 직접 연락 가능한 식별 정보는 의도적으로 포함하지 않습니다. 보유기간: 오류 진단 목적 달성 시까지이며, 세부 보관 기간은 Sentry 자체 정책에 따릅니다
• Google LLC (미국, https://cloud.google.com/terms/cloud-privacy-notice): 커뮤니티에 업로드되는 사진·영상의 콘텐츠 자동 검수(음란물 등 명백히 부적절한 콘텐츠의 게시 전 차단)를 위해 Google Cloud Vision API·Video Intelligence API 를 사용합니다. 업로드된 이미지·영상이 게시 전 검수 목적으로 이전·처리되며, 검수용 임시 저장분은 처리 직후 삭제됩니다. 보유기간: 검수 처리 완료 시까지

사용자는 국외 이전을 거부할 수 있으나, 이 경우 커뮤니티 업로드, 오류 리포팅, 자동가입 방지 등 서비스의 전부 또는 일부 이용이 제한될 수 있습니다. 거부 또는 삭제 요청은 iconic.app.official@gmail.com 로 문의할 수 있으며, 접수 후 지체 없이(특별한 사정이 없는 한 10일 이내) 처리 결과를 안내합니다. 커뮤니티에 업로드되는 사진·영상은 게시 전 Google Cloud 자동 검수를 거칩니다. 반면 심령 도구의 사진 분석(판독)은 전부 기기 내부(온디바이스)에서 처리되며 외부로 전송되지 않습니다.

7. 제3자 제공

서비스는 사용자의 정보를 제3자에게 판매·공유하지 않습니다. 도구의 사진 분석은 전부 기기 내부에서 이뤄져 사진이 외부 AI로 전송되지 않습니다. 법령에 따른 정부 요청이 있는 경우에만 협조할 수 있습니다.
